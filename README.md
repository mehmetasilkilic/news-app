# News App Innoscripta

## Run Locally

Clone the project

```bash
  git clone https://github.com/mehmetasilkilic/news-app.git
```

Go to the project directory

```bash
  cd news-app
```

Run docker compose

```bash
  docker-compose -f docker-compose.dev.yml up
```

Open localhost

```bash
  http://localhost:3000/
```
